# Admin Dashboard

This project is a web-based admin panel for monitoring and managing air quality data.  
It uses PHP and MySQL to insert and retrieve sensor readings from an ESP32 device.


# لوحة تحكم الإدارة

هذا المشروع عبارة عن لوحة تحكم لعرض وقراءة بيانات جودة الهواء القادمة من جهاز ESP32.  
يستخدم PHP وMySQL لإرسال واستقبال القراءات من الحساسات.
